### Описание:

CRUD на Angular с использованием JSON-сервера. Добавление новых и удаление уже существущих пользователей, а так же редактирование информации о них.

### Стек:

- Angular 17;
- JSON-Server;
- Bootstrap;
- Bootstrap Icons.

### Основные команды:

- npm i bootstrap
- npm i bootstrap-icons
- npm i json-server
- npx json-server --watch db.json
- ng serve
